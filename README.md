# Gift Card Redemption System

A Dockerized backend system for issuing and redeeming gift cards securely.

## Tech Stack
- Backend: FastAPI (Python)
- Database: PostgreSQL
- Cache: Redis
- Frontend: HTML, CSS, JS
- Containerization: Docker

## Features
- Gift card issuance
- Secure validation
- Partial/full redemption
- Concurrency-safe balance updates

## Setup

```bash
docker-compose up --build
